# 12-image-security — Image Security

Herramientas para escanear vulnerabilidades en imágenes, generar SBOMs y verificar la integridad mediante firmas.

## Herramientas

- [Trivy](./01-trivy/README.md) — Vulnerability scanner / CLI · CNCF Graduated · Estable
- [Grype / Syft](./02-grype-syft/README.md) — Scanner / SBOM generator · Not CNCF · Estable
- [Clair](./03-clair/README.md) — Vulnerability scanner · Not CNCF · Estable
- [Cosign / Sigstore](./04-cosign-sigstore/README.md) — Image signing / CLI · CNCF Graduated · Estable
- [Notary v2 / Notation](./05-notary-v2-notation/README.md) — Image signing · CNCF Incubating · Estable
- [Kyverno (verify images)](./06-kyverno-verify-images/README.md) — Policy / Admission webhook · CNCF Graduated · Estable
- [Connaisseur](./07-connaisseur/README.md) — Admission controller · Not CNCF · Estable
