# 32-api-gateways-and-management — API Gateways & Management

API gateways puros y plataformas de API management para publicar, proteger, enrutar, monetizar y monitorizar APIs sobre Kubernetes.

Cubre desde gateways de alto rendimiento y bajo overhead (KrakenD, APISIX) hasta plataformas completas de management con portal de desarrolladores, analíticas y monetización (Kong, Tyk, Apigee, Gravitee, 3scale).
