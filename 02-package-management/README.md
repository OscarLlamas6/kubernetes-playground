# 02-package-management — Package Management

Herramientas para empaquetar, versionar y desplegar aplicaciones en Kubernetes de forma reproducible.

## Herramientas

- [Helm](./01-helm/README.md) — Package manager / CLI · CNCF Graduated · Estable
- [Kustomize](./02-kustomize/README.md) — Template-free config · Not CNCF (k8s built-in) · Estable
- [Carvel (ytt/kapp/kbld)](./03-carvel-ytt-kapp-kbld/README.md) — CLI tools · CNCF Sandbox · Estable
- [Jsonnet / Tanka](./04-jsonnet-tanka/README.md) — Config language / CLI · Not CNCF · Estable
- [cdk8s](./05-cdk8s/README.md) — CDK / Code-based · CNCF Sandbox · Estable
- [Timoni](./06-timoni/README.md) — Package manager / CUE · Not CNCF · Beta
- [Glasskube](./07-glasskube/README.md) — Package manager / UI · Not CNCF · Beta
